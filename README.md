# 🎯 Job Monitor

Automated job tracking for Data Science, ML, and AI roles across target companies.

## Dashboard

Open `index.html` in a browser to view the dashboard locally, or visit the GitHub Pages version.

## How It Works

1. **Scanner** (`scanner.py`) crawls career pages for configured companies
2. **Database** (`jobs-db.json`) stores all discovered jobs with timestamps
3. **Dashboard** (`index.html`) displays jobs with filters for relevant/new
4. **GitHub Actions** runs daily at 9 AM EST

## Usage

### Manual Scan (All Companies)
```bash
source venv/bin/activate
python scanner.py
```

### Scan Single Company
```bash
python scanner.py visa
python scanner.py deloitte
python scanner.py cisco
```

### Adding a New Company

Edit `config.json`:

```json
{
  "companies": [
    {
      "id": "company-id",
      "name": "Company Name",
      "url": "https://careers.company.com/jobs",
      "referrers": ["Contact Name"],
      "added": "2026-03-01"
    }
  ]
}
```

Then add a fetcher function in `scanner.py` and register it in the `FETCHERS` dict.

## Files

- `config.json` - Target roles and company configs
- `jobs-db.json` - Job database (auto-updated)
- `run-log.md` - Scan history log
- `scan-summary.md` - Latest scan summary (generated by GitHub Actions)
- `index.html` - Dashboard UI

## GitHub Actions

The scanner runs automatically via GitHub Actions:
- **Schedule:** Daily at 9 AM EST (14:00 UTC)
- **Manual:** Can be triggered from Actions tab

Results are committed back to the repo automatically.
